## Main Goal
Reach the target Wikipedia page from the start page by clicking only on links in the article text.

---

## Single Player
Play locally and try to get from start to target as fast as possible or with as few clicks as possible.

## Local Game Modes

### Set Run
Player can set their Start or target pages or both.
If only other was is set the other one is randomly generated.

### Random
Both start and target pages are randomly generated.
Completing the run in this game mode saves stats from your run to the [Leaderboard](#leaderboard).

### Time Attack
Same as [Set Run](#set-run) game mode with time limit that can be either Standard 5 minutes or user set custom time limit between 1 to 60 minutes.

### Scoring
- **Clicks count**: Each link clicked increases your click count.
- **Time bonus**: Completing faster earns a higher score.

---

## Multiplayer

### Objective
Race against in or with other players to reach the same target page first.

### Rules
1. Players start from the same page and target.
2. Only links in the article text may be clicked.
3. Your **current page and click count** are visible to other players (if enabled).
4. Multiplayer can be:
   - **Live**: Compete in real time.
   - **Turn-based**: Take turns if using asynchronous mode.

### Scoring
- **First to target**: Wins the round.

---

## Game Mechanics

### Leaderboard
- [Random](#random) game mode: Completing run saves your username, total clicks, start, target pages and completion time to the Random ranked leaderboard.
- [Time Attack](#time-attack) game mode: Completing run with random start and target pages in standard time limit saves the same statistics as Random ranked leaderboard but in place of completion time it logs the time left.

### Party
- Create party with "Create Party" button on main menu.
   - You are taken to party lobby with your party code
- Join party with "Join Party" button on main menu.
   - When asked enter party code
